# Q12. Microcontroller vs Microprocessor

| Parameter | Microcontroller (Arduino UNO) | Microprocessor (Raspberry Pi 4) |
| --- | --- | --- |
| CPU Speed | 16 MHz | 1.5 GHz to 1.8 GHz |
| On-chip RAM | 2 KB (SRAM) | None (Uses external 2GB / 4GB / 8GB LPDDR4) |
| On-chip Flash | 32 KB | None (Uses external MicroSD card or eMMC) |
| Operating System | None (Runs bare-metal firmware loops) | Full OS (e.g., Linux / Raspberry Pi OS) |
| Typical Use Case | Reading single sensors, controlling relays, blinking LEDs | Media streaming, edge AI computing, web hosting |
| Power Consumption | Extremely low (~20-50 mA, Milliwatts) | High (~500-1200 mA, up to 15 Watts) |
