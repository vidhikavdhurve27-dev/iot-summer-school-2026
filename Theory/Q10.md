# Q10. (Theory) What is a .gitignore file? Write a .gitignore for an Arduino project that excludes compiled output files (.hex, .elf), OS-specific files (.DS_Store, Thumbs.db), and IDE config folders (.vscode/, build/). (2 marks)
## Definition:
### A *.gitignore* file is a plain text file that tells Git which files or folders to permanently ignore. It keeps repositories clean by stopping temporary, large, or compiled files from being tracked or uploaded to GitHub.

## Required Arduino .gitignore File
### Compiled Output Files
*.hex
*.elf

### OS-specific Files
.DS_Store
Thumbs.db

### IDE Config Folders
.vscode/
build/
